Shopping Cart Project

In this project I created a interactive shopping cart. You can add items to your cart which will remove the item from inventory, it updates your total and you can remove items you no longer want. With the use of an Amazon AWS database you can reload the store when you run it locally. If you want to try it out head over to the git repo where you can clone the code and create your own.

Three Tier Bank Website

This project is a three tier banking application. You can create an account, login, deposit, withdraw, and check your balance. You can click on the link to see the site in action or for more accessibility and control head to the repo where you can clone all the files and get started.

Restaurant App

This is a three tier full stack application that is a restuarnat delivery app. You can choose restaurants and add dishes to your cart that will update with changes. After decided on what you want to order you can checkout using stripe. All users, orders, restaurants are tracked in Strapi

Real Time Bus Tracker

For this project I used mapbox to create a real time tracker of busses moving between MIT and Harvard. This map also has icons for each of the bus stops between the two schools. If you want to try it out you can see the project in the browser by clicking on the image or click on the link to find the code so you can create your own.

PacMan Game

This project I created a PacMen factory where you can click a button to add as many new PacMen as you want. After that you can start movement where they will move randomly through out the screen and will bounce off all the edges. If you want to try it out you can see the project in the browser by clicking on the image or click on the link to find the code so you can create your own.
