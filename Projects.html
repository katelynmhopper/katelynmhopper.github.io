This project I created a PacMen factory where you can click a button to add as many new PacMen as you want. After that you can start movement where they will move randomly through out the screen and will bounce off all the edges.
This exercise I created two eyes that would follow your mouse movements on the screen.
For this project I used mapbox to create a real time tracker of busses moving between MIT and Harvard. This map also has icons for each of the bus stops between the two schools. To get this to run you will need to insert your own access token from mapbox.
